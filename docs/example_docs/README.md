# Example Docs

> Completed reference versions of every template in this repo.
> All examples use the same fictional project — **OrderFlow API** — so the cross-references
> between files make sense as a coherent whole.

**Fictional project used throughout:**
- **Project:** OrderFlow API — an order management REST API for an e-commerce platform
- **Org:** `acme-corp`
- **Stack:** TypeScript · Node.js 22 · Fastify · PostgreSQL · Drizzle ORM
- **Deployment:** Railway (API + DB) · Vercel (storefront frontend)
- **Team:** Backend Engineering (`#eng-backend` on Slack)
