# template-cf-pages-static

Starter repository for static or mostly static public sites with light edge logic.

## Best fit
- marketing sites
- SEO pages
- landing pages
- docs sites
- simple forms and webhook receivers

## Includes
- static public site
- Cloudflare Pages Functions
- security middleware
- `wrangler.jsonc`
- CI and deploy workflows
- cache and environment docs

## Quick start
1. Install dependencies.
2. Copy `.dev.vars.example` to `.dev.vars` for local development.
3. Run `npm run dev`.
4. Configure Cloudflare secrets and repo variables before enabling deploys.
